🎉 Todo List Website - Local Viewer

Note: To view the website locally, click one of the buttons below:
📝 Open Todo List 🏆 Open Accomplishments

Features Available:

  • ✅ Add and manage todo items
  • 💾 LocalStorage persistence
  • 🏆 Move tasks to accomplishments
  • 🧮 Working calculator
  • 📝 Auto-save notepad
  • ⏰ Live date & time

To Deploy Online:

  1. Open PowerShell in the project folder
  2. Run: npm run deploy
  3. Visit: https://Shiroe28.github.io/personaltodolist
Alternative: Use VS Code Live Server extension or Python's HTTP server to view the site properly.
